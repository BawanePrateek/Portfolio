DLC & PME – Project Highlights

Rack‑Level DLC (150 kW) – CDU Software & Controls

  • Defined software specifications for Commissioning, Autofill, Bleeding, Manual, Boot, Purge, Hot Swap, Fail Safe modes.
  • Sensors Temperature, Pressure, Flow, Leak, Level.
  • Condensation protection via dew point (ambient T + RH) & dynamic secondary supply setpoint.
  • Comprehensive alarms: TA1/TA2, PA1/PA2/PA3/PA4, PUA1‑4, SVA1‑4, LA1, ASA1.
  • Defined events and conditions that should be recorded in the log file for data logging.
  • Data retention is dynamically determined based on the volume of generated events and the size of each message, ensuring efficient storage and timely rollover.
  • Rollover mechanism with three files of 500KB each, on full capacity, oldest is deleted, others shift forward, and a new file is created at the third position to maintain continuous logging.
  • Test Plans tool for structured test case management.
